## how to run server
1. first, clone this repo.

`git clone https://github.com/teang1995/movie_review_project.git`

2. move to our project folder

`cd movie_review_project`

3. dockerfile build

`docker build -t spring_docker_test:v0.1 .`

4. run server

`sh run_server.sh`
